# Worker Rights CN

[简体中文](README.md) | [English](README.en.md) | [项目网站](https://90le.github.io/worker-rights-cn/)

面向中国大陆普通劳动者的开源权益助手：用普通中文整理事实、证据、金额估算和文书草稿。Codex 是规范入口，其他宿主只是薄适配器。

## 紧急与安全边界

如果你今天被要求立即签字、正在工伤就医、面临仲裁时效，或安全受到威胁：先保留原件和时间线，不要在未理解的文件上签字，并及时联系当地人社部门、工会或持证律师。

本项目不是律师、律师事务所、政府服务或自动裁决工具；不提供个案代理，不保证协商、仲裁或诉讼结果。所有估算和草稿都需要你核对事实、当地规则与最新来源。

## 30 秒 Codex 开始

1. 运行 `codex plugin marketplace add 90le/worker-rights-cn --ref main`。这只是添加 marketplace 来源，不会直接安装插件。
2. 运行 `codex plugin add worker-rights-cn@worker-rights-cn` 完成安装。
3. 或者重启/刷新 Codex/ChatGPT 桌面端，在 **Plugins** 中选择 **Worker Rights CN** marketplace 后安装。

安装后不必先学法律术语，直接讲发生了什么。

### 第一条提示词

> 我在【城市】工作，公司今天【发生的事】。请先区分已确认事实和待补充信息，告诉我现在先不要做什么、今天应保存什么，再列出可能权益和下一步。不要保存或上传我的材料。

## 能做什么 / 不能做什么

能做什么：整理时间线和待确认事实；提醒优先保存的证据；显示补偿计算式和假设；审查协议风险；准备沟通、协商和仲裁草稿；附来源并标注需当地确认的结论。

不能做什么：替你签字、提交、上传或联系第三方；自动保存聊天或个案材料；代表律师、政府或任何官方机构；将估算当成裁决结果。

## 四步流程

1. **讲情况**：用自己的话说明发生了什么。
2. **补信息**：核对日期、工资、岗位、通知和证据。
3. **看建议**：查看事实、估算、来源、风险和可选行动。
4. **你来决定**：先核对，再选择是否保存、导出或寻求专业帮助。

## 五个常见场景

- **公司欠薪**：整理应发、实发、发薪日和沟通记录。
- **突然被辞退**：核对理由、方式、时间和工资基数。
- **今天要签协议**：检查解除原因、金额、付款时间、权利放弃和违约条款。
- **加班记录不全**：从考勤、排班、聊天、邮件、门禁和工作成果互相印证。
- **准备劳动仲裁**：按请求、事实、证据和来源组织草稿。

示例均为虚构且已脱敏，不对应任何真实个人或案件。

## 隐私

- 默认不自动保存、不自动上传、不自动对外发送。
- 需要落盘时，应先说明绝对路径和保存范围，并获得明确同意。
- 导出前检查姓名、证件号、电话、住址、银行账号等个人信息。
- 详见[隐私政策](PRIVACY.md)、[隐私与本地存储](docs/%E9%9A%90%E7%A7%81%E4%B8%8E%E6%9C%AC%E5%9C%B0%E5%AD%98%E5%82%A8.md)。

## 安装、更新、卸载与数据保留

- **安装**：先添加 marketplace，再运行 `codex plugin add worker-rights-cn@worker-rights-cn`；也可以刷新桌面端后在 Plugins 中安装。marketplace-add 本身不是安装命令。
- **更新**：运行 `codex plugin marketplace upgrade worker-rights-cn`，然后刷新桌面端并按界面提示更新或重新安装。
- **卸载**：在 Plugins 中卸载 Worker Rights CN。卸载插件默认不代表删除你曾明确保存的用户数据。
- **彻底清除**：先确认不再需要导出材料，再在宿主展示的数据目录中删除 `worker-rights-cn` 目录，并复查目录、索引与关联审计记录。这是不可恢复操作，必须明确确认。
