# 如何审查协议

先看协议内容，再决定是否签。不要只看标题和补偿数字。

以下示例均为虚构，已脱敏，不对应任何真实个人或案件。

## 现在先不要做什么

- 不要在空白页或附件缺失时签字。
- 不要只听口头解释。
- 不要忽略“已结清”“自愿离职”“放弃权利”“违约金”“保密”“竞业”等条款。
- 不要把整个含真实个人信息的协议上传到不明服务。

## 今天应当保存什么

保存完整协议、全部附件、版本时间、发送人、签署期限和对方口头承诺。

如需让插件审查，先遮盖姓名、联系方式、证件号、住址、银行账号和无关商业秘密。保留一份未修改原件在自己控制的位置。

## 当前可能涉及哪些权益

可以这样问：

> 请审查这份虚构的离职协议摘要。逐条说明付款时间、税费、解除原因、权利放弃、保密、竞业、违约责任和争议解决。不要替我确认签署。

插件应区分：原文事实 `confirmed_fact`、条款风险 `supported_assessment`、需要当地确认的事项 `local_verify`、签署前应复核的高风险条款 `lawyer_review`。

它可以生成修改建议和向 HR 追问的文字，但不会替你签署或发送。

## 下一步需要补充什么信息

提供协议全文或相关条款、附件是否齐全、付款安排、解除日期、工作地、竞业安排和你最在意的目标。

签字会造成重要权利变化时，保留核对时间，并考虑由执业律师复核。
