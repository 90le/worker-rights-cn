# 如何准备劳动仲裁

仲裁材料应从事实和证据出发，不是先写一篇情绪陈述。

以下示例均为虚构，已脱敏，不对应任何真实个人或案件。

## 现在先不要做什么

- 不要编造日期、金额或对话。
- 不要把互相冲突的请求简单堆在一起。
- 不要忽视期限和受理地点。
- 不要在未核对前提交自动生成的草稿。

## 今天应当保存什么

准备劳动关系材料、工资记录、争议通知、完整沟通记录、解除文件和已付款凭证。

建立一条时间线。每个事件写日期、参与人、发生内容、对应材料和仍有争议的事实。

## 当前可能涉及哪些权益

插件可以帮助整理：

- 当事人和劳动关系信息。
- 仲裁请求草稿及计算过程。
- 事实与理由草稿。
- 证据目录及每份证据的证明目的。
- 缺失材料和由对方控制的材料。
- 来源锚点与需要当地核实的事项。

受理地点、当地表格和特定数值通常需要 `local_verify`。请求设计和提交前检查可能需要 `lawyer_review`。

## 下一步需要补充什么信息

告诉 Codex：工作地、公司主体、合同主体、争议发生时间、你的请求目标、已知金额、现有证据和是否收到书面解除文件。

可以输入：

> 请根据虚构劳动者甲的脱敏材料生成仲裁准备清单。先列缺失事实和证据，不要直接生成可提交的最终文件。

提交前，逐项核对姓名、主体、日期、金额、请求之间的关系、证据编号和当地要求。
