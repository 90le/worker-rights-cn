# 裁员前后 72 小时

这份清单适合突然收到裁员、辞退或离职要求的人。

以下示例均为虚构，已脱敏，不对应任何真实个人或案件。

## 现在先不要做什么

- 不要因为“今天必须签”就立即签字。
- 不要主动写下与你真实意思不一致的“个人原因离职”。
- 不要删除聊天、邮件或文件的原始版本。
- 不要拿走无权访问的公司资料。
- 不要威胁、辱骂或公开无关人员信息。

如有人身安全、急性健康或自伤风险，先联系当地紧急服务和可信赖的人。劳动材料可以稍后整理。

## 今天应当保存什么

只保存你合法接触且与本案有关的材料：

- 劳动合同、续签记录、员工手册签收记录。
- 工资单、银行入账、个税和社保记录。
- 裁员、辞退、调岗、绩效或谈话通知。
- 与 HR、主管沟通的完整上下文。
- 门禁、考勤、工作成果等能证明劳动关系和工作情况的材料。
- 每次谈话的时间、地点、参加人和原话摘要。

保留原件。另做只读副本。记下取得方式。录音和拍摄是否合法需要结合场景核对。

## 当前可能涉及哪些权益

插件会区分解除类型、程序、补偿路径、工资和未休假等项目。

结果可能是 `supported_assessment`、`estimate` 或 `local_verify`。这不是胜诉保证。

以下虚构说法可直接输入：

> 我在广州工作。公司口头说岗位取消，要求两天内签协议。我想先判断解除类型、证据缺口和补偿估算。现在不要保存案件。

## 下一步需要补充什么信息

先准备这些事实：

1. 实际工作地和公司注册地。
2. 入职、合同到期和通知日期。
3. 最近工资构成与发放记录。
4. 公司给出的理由和文件原文。
5. 是否孕期、医疗期、工伤或存在其他特殊保护情况。
6. 你最紧急的目标：争取时间、继续履行、补偿谈判或准备仲裁。

继续阅读：[如何整理证据](如何整理证据.md) 和 [如何估算补偿](如何估算补偿.md)。
