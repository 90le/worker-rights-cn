# 支持方式

## 项目问题与社区交流

可使用 GitHub Issue 报告可复现的程序错误、提出功能建议或文档改进。先搜索已有 Issue，并只提供虚构、彻底脱敏的最小复现。不要提交真实个案证据或敏感个人信息，包括真实姓名、身份证号、手机号、合同、聊天记录、录音、病历或银行卡信息。安全漏洞须按 [SECURITY.md](SECURITY.md) 私密报告。

微信 `binstudy` 仅用于项目交流、贡献协作和社区联络，不提供一对一法律代理，不建立律师委托关系，也不承诺回复时限或个案结果。

## 紧急或个人法律帮助

本仓库的 Issue、维护者和社区渠道不是紧急法律服务，也不适合提交个案材料。若面临今天必须签字、仲裁或诉讼时效临近、人身安全、强迫、欠薪影响基本生活等紧急情况，请尽快联系当地人社部门、工会、法律援助机构、执业律师或紧急服务。请在安全的专业渠道中自行核验对方身份后再提供必要材料。

Worker Rights CN 提供信息整理与一般性辅助，不是“AI 律师”，不保证结果，也不代表任何官方平台。
